Changelog
=========

1.2.0
-----

* Added Discord integration.

1.1.0
-----

* Withers, dragons, illusioners, giants, elder guardians, and wardens will now have death messages by default.

1.0.0
-----

* Added option to display death messages for any entity type.
  The death messages can be displayed, or not displayed based on the following factors:
  * Whether the entity is tamed
  * Whether the entity has a custom name
  
  Alternatively, death messages can be displayed for all entities of the given type.
* Added a default configuration for entities. This will be used if there is no entity-specific configuration.
* Added option for the death location to be included in the messages.
* By default, villagers, wandering traders, and named entities will have their death messages broadcast to all players.
